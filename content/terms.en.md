## 4. Supported scope

Unless otherwise stated on the product page, the current intended scope of HardenLab Local Pack is limited in particular to:

- Windows 11 Pro and Enterprise
- primary target release: Windows 11 25H2
- Windows 11 24H2 also supported
- languages: de-DE and en-US
- local deployment
- environments without Active Directory

The current core scope focuses in particular on:

- Microsoft Defender
- Firewall
- UAC / LSA-related settings
- Windows Update

Unless explicitly stated otherwise, the following are outside the supported scope in particular:

- RDS environments
- Windows Home

- Intune / MDM / Autopilot
- centrally managed zero-touch rollout
- Office, Microsoft 365, or Adobe hardening
- production-critical specialty software outside the documented support scope
