# HardenLab

**HardenLab Local Pack**

Windows-Haertung ohne Active Directory.

HardenLab Local Pack ist ein konservatives, lokal ausrollbares Hardening-Paket fuer **Windows 11** in **Standalone-, Workgroup- und kleinen lokalen Umgebungen ohne Active Directory**.

## Wofuer das Produkt gedacht ist

- kleine Firmen und KMU ohne eigene IT-Abteilung
- lokal administrierte Einzelgeraete
- kleine Umgebungen ohne AD, Intune oder MDM
- Teams, die eine supportbare Grundhaertung statt eines komplexen Management-Stacks brauchen

## Wofuer es nicht gedacht ist

- Active Directory / Domain GPO
- Intune / MDM / Autopilot
- Server, RDS, Domain Controller
- Office-, Adobe- oder Flotten-Hardening ausserhalb des dokumentierten Scopes

## Aktueller v1-Scope

- Windows 11 Pro und Enterprise
- 25H2 primaer, 24H2 weiterhin unterstuetzt
- de-DE und en-US
- Deployment lokal via LGPO.exe
- Kernbaseline: Defender, Firewall, UAC/LSA, Windows Update
