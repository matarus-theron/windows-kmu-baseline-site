# Windows KMU Baseline Pack

A technically clean, modular hardening and policy package for **Windows 11 Pro** in **small environments without Active Directory**.

## Focus of v1

- Windows 11 Pro
- 24H2
- de-DE and en-US
- local deployment via LGPO.exe
- Defender
- Firewall
- UAC / LSA
- Windows Update
