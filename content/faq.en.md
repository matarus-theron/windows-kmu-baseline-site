# FAQ

## Is this meant for Active Directory?
No. The current v1 product is **HardenLab Local Pack** for environments **without Active Directory**.

## Does it work on standalone devices?
Yes. The scope covers standalone systems and small local environments.

## Is audit included in the core product?
No. Audit is planned as a separate module / add-on.

## Is browser hardening included?
Not in the core baseline. Browser hardening is planned as a separate add-on.
