# Support-Matrix

## Offiziell unterstützt

- Windows 11 Pro
- Version 24H2
- de-DE
- en-US
- lokales Deployment via LGPO.exe
- Workgroup / Einzelgeräte / kleine Umgebungen

## Best Effort / optional

- Windows 11 Enterprise
- Windows 11 25H2 nach Tests
- Browser-Hardening als Add-on
- konservative SMB-/Netzwerk-Härtung
- optionale Edge-/SMB-Sections außerhalb der Kernbaseline

## Nicht unterstützt / nicht empfohlen

- Windows 11 Home
- Server 2022 / 2025
- Active Directory / Domain GPO
- Domain Controller / RDS
- Intune / MDM / Autopilot
- Office / M365 Hardening
- Adobe / Acrobat Hardening
- Zero-Touch / zentraler Flotten-Rollout
- produktionskritische Spezialsoftware außerhalb des Support-Scopes
