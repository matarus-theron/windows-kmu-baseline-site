# Delivery

This page describes the planned delivery and package availability framework of the **Windows KMU Baseline Pack**.

{{< callout type="important" >}}
A public direct download is not yet enabled. This page serves as structured orientation for the future delivery model.
{{< /callout >}}

## Planned approach

- a clearly scoped product package
- documented product boundaries
- technical accompanying documentation
- focus on local deployment in small Windows 11 environments without Active Directory :contentReference[oaicite:5]{index=5}

## Buyers should later receive

- the product package in the selected edition
- accompanying documentation
- scope and usage guidance
- pilot-device and controlled rollout guidance
- legal and product-related supporting information :contentReference[oaicite:6]{index=6}

## Not intended as

- open anonymous mass download
- uncontrolled immediate rollout without review
- domain-/AD-centric delivery
- zero-touch or fleet deployment within v1 scope :contentReference[oaicite:7]{index=7}
